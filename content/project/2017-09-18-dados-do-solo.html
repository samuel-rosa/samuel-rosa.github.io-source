A pesquisa pedológica já produziu um volume incomensurável de dados do solo. Muitos desses dados já podem ter sido perdidos para sempre pela falta de infraestuturas para seu armazenamento e distribuição. A situação não é diferente com os milhares de novos registros produzidos a cada dia que passa. Preocupado com isso, venho desenvolvendo atividades diversas relacionadas à construção de tais infraestruturas, o que deu origem ao projeto

  • Infraestruturas públicas de dados do solo – da concepção à distribuição

Atualmente, as principais atividades desse projeto são:

  1. Controle de qualidade do Sistema de Informação de Solos Brasileiros (link);
  2. Construção da Base de Dados Brasileira de Ferro do Solo (link)
